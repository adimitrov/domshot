# domshot
=======

## Setup
Install PhantomJS.
```
brew install phantomjs
```

## Usage
```
phantomjs domshot.js <some URL> <some selector> <destination.png>
```

Example:

```
phantomjs domshot.js http://google.com .news output.png
```
